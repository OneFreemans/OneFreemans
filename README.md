Асинхронный Telegram-бот с поддержкой:

#### 🎧 [Визуализатор направления звука](https://github.com/OneFreemans/sound-direction-visualizer)
Утилита для визуализации направления звука в реальном времени. 
Создана для компенсации недостатка слуха в играх.
Можете не переживать за блокировку игрового аккаунта, утилита не как не взаимодействует с файлами игр.
Проверено в течение года на примере игры EFT.
**Сгенерирована за 1 промт с помощью Giga Agent** — пример эффективного использования ИИ-инструментов.

#### 🔐 [Менеджер паролей](https://github.com/OneFreemans/password-manager)
Локальный менеджер паролей с защитой мастер-паролем и шифрованием данных:
- хеширование мастер-пароля (bcrypt), шифрование паролей сайтов (Fernet, cryptography)
- защита от брутфорса: ограничение попыток, временная блокировка, полное удаление данных
- хранение в SQLite, консольный интерфейс, чистая архитектура
